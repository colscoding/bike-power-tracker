# Fitness Tracker Apps - Complete Features List

## TrainingPeaks

### Core Features
- **Activity Tracking & Analysis**: Complete workout logging with detailed metrics analysis
- **Performance Management Chart (PMC)**: Track Fitness (CTL), Fatigue (ATL), and Form (TSB)
- **Training Stress Score (TSS)**: Quantify workout intensity and training load
- **Structured Workout Builder**: Create detailed interval-based workouts with power/pace/HR targets
- **Strength Builder**: Build structured strength workouts with exercise library and video demonstrations
- **Calendar & Planning**: Comprehensive training calendar with compliance tracking
- **Peak Performances**: Track personal bests across different time durations
- **StackUp**: Compare power/pace performance to other TrainingPeaks users by age and gender
- **Apple Watch Integration**: Sync structured workouts directly to Apple Watch
- **TrainingPeaks Virtual**: Indoor cycling platform with realistic physics, GPXplore routes, and group rides
- **Recurring Workouts**: Schedule workouts to repeat automatically for up to 90 days
- **Equipment Tracking**: Monitor mileage on bikes and shoes
- **Weather Integration**: View local weather forecasts in calendar
- **MyFitnessPal Integration**: Sync nutrition data
- **Overlay Charts**: Compare any data channels on same chart
- **Sports Subtypes**: Track multiple sport variations
- **Multi-language Support**: German, Portuguese, Spanish variants
- **Threshold Detection**: Automatic FTP/threshold change notifications
- **HRV & Resting HR Sync**: Automatic sync from Garmin devices
- **Limited Availability**: Athletes can mark time/equipment constraints
- **Athlete Groups**: View and plan for multiple similar athletes simultaneously
- **Goal Setting**: Track process goals beyond events (hydration, sleep, mobility)
- **Weekly Snapshot**: See planned vs completed workouts by duration, distance, or TSS

### Device Compatibility
- Garmin, Suunto, Wahoo, Polar, Coros, Apple Watch
- 100+ compatible apps and devices

## Runalyze

### Core Features
- **Effective VO2max Estimation**: Calculate VO2max from every activity based on HR and pace
- **Race Predictions**: Multiple models (CPP, Steffny, Cameron, VDOT) for distances from 3k to ultra-marathon

- **Training Load Management**: Heart rate-based TRIMP calculation for overall load
- **Acute & Chronic Training Load**: Monitor CTL and ATL trends
- **Climb Score & Categorization**: Categorize climbs from 5th category to HC using FIETS index
- **Streak Analysis**: Track daily running streaks
- **Equipment Tracking**: Monitor mileage and replacement needs for shoes and bike parts
- **Activity Type Prediction**: Machine learning auto-detection of workout types
- **Recurring Routes**: Automatic route matching and performance comparison
- **Pace Recommendations**: Configure custom training paces
- **Quality Sessions Analysis**: Deep dive into tempo training efforts
- **Performance Curves**: 90-day power, HR, and pace analysis
- **Competition Performance Chart**: Compare race results by age grade, VO2max, or speed
- **Trend Analysis**: Boxplots, dynamic histograms, and data tables
- **HRV Data**: Import and analyze heart rate variability
- **Running Dynamics**: Support for advanced running metrics
- **Poster Generator**: Create heatmap, grid, circular, or calendar posters
- **300+ Metrics**: Track extensive performance and physiological markers
- **Aerobic Threshold Estimation**: Using HRV data
- **Training Monotony Calculator**: Assess training variety
- **Tiles Counter**: Premium analysis tool
- **VO2max Factor Analysis**: Premium feature for detailed performance breakdown

### Import/Export
- **File Format Support**: .fit, .fitlog, .gpx, .hrm, .kml, .logbook, .pwx, .slf, .sml, .tcx, .trk, .ttbin, .xml
- **Direct Sync**: Garmin Connect, Polar Flow, Suunto App, Fitbit, Rouvy, Hammerhead, Concept2
- **Export**: Single activities as .tcx, .gpx, .kml, .fitlog

### Sharing & Privacy
- Privacy-first approach, all settings default to private
